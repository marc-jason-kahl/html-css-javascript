<HTML>

Um elektronische Dokumente wie Texte, Hyperlinks (Links zu anderen Seiten), Bildern usw. zu strukturieren benutzt man HTML (Hypertext Markup Language). Diese Auszeichnungssprache ist die Grundlage für das World Wide Web.

HTML wird durch verschiedene Tags (Auszeichnungen) so wie <head> und <body> strukturiert. Der <head> beinhaltet alle nicht sichtbaren Einstellung der Website. Der <body> hingegen ist dazu da um Informationen die sichtbar sind, also die Inhalte der Seite, die der User sieht, darzustellen.

Der <head> kann z.B. das Meta Tag enthalten. Ein Beispiel hierfür wäre das Meta Tag um die Website auf eine UTF-8 Codierung einzustellen. Des Weiteren werden hier auch die Open Graph Tags für Social Media Informationen angegeben und in den meisten Fällen Externe Dateien wie CSS oder JavaScript verlinkt.

Einige Beispiele für den <body> währen der h1 Tag, der die Überschrift darstellt oder das p Tag, dass zur Darstellung von Paragraphen da ist. Hier gibt es noch etliche weitere HTML Tags wie z.B. Formulare, Listen, Buttons, usw.

<html>

  <head>
  <!-- Head content -->
  </head>

  <body>
  <!-- Body content -->
  </body>
</html>

CSS

CSS ist eine Computersprache um Websites aufzubauen und zu struckturieren. Diese Sprache enthält Codeelemente und setzt sich aus diesen "Cascading Style Sheets" zusammen, die auch CSS-Dateien (.css) genannt werden.

Es gibt ID's und Klassen um die Element im Dokument zu identifizieren. ID's sind dazu da, um einzellne Elemente zu bestimmen:

#beispiel {}

während Klassen dafür da sind, mehrere Elementen zu bestimmen.

.beispiel {}

Man kann auch alle Elemente eines Tags (z.B. alle divs) bestimmen.

div {}

Oder diese Kombinieren.

div.beispiel {}
.beispiel#beispiel {}

JavaScript

JavaScript gehört heutzutage mit zu den Kernkomponent einer Website. Es gibt diverse Anwendungsgebiet hierfür wie z.B. Client-Seitiges Javascript um die Website interaktiver zu gestallten oder Elemente dynamisch nach zu laden.

Um ein paar bekannte Einsatzmöglichkeiten, die jeder kennt, zu nennen, wären hier Seiten die "infinity scrolling" nutzen. Der Benutzer scrollt die Seite und es werden kontinuierlich Elemente wie z.B. Bilder nachgeladen.

Andere Einsatztgebiete sind Lightboxen, Slider oder Filter.

Im Gegensatzt zu CSS und HTML ist JavaScript eine vollwärtige Programmiersprache.

// Eine simple for Schleife
for(const i = 0; i < 10; i++) {
  console.log(i)
}

// Ein If Conditional
const x = 4
if(x === 4) {
  console.log("x is gleich 4")
} else {
  console.log("x is ungleich 4")
}

JavaScript findet man heutzutage aber nicht nur Anwendung im Frontend einer Website sondern auch in vielen anderen Gebieten wie in SPAs, im Backend von Webseiten oder in Shell Scripten.
