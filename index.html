<HTML>

Um elektronische Dolkumente wie Texte, Hyperlinks (Links zu anderen Seiten), Bildern usw. zu struckturieren benutzt man HTML (Hypertext Markup Languag). Diese Auszeichnungssprache ist die Grundlage für das World Wide Web.

HTML wird durch verschiedene Tags (Auszeichnungen) so wie <head> und <body> strukturiert. Der <head> beinhaltet alle nicht sichtbaren Einstellung der Website. Der <body> hingegen ist dazu da um Informationen die sichtbar sind, also die Inhalte der Seite die der User sieht, darzustellen.

Der <head> kann z.B. das meta Tag enthalten. Ein beispielt hierfür wäre das meta tag um die Website auf ein UTF-8 Codierung einzustellen. Desweiteren werden hier auch die Open Graph Tags für social media Informationen angegeben und in den meisten fällen Extern Dateien wie CSS oder JavaScript verlinkt.

Einige Beispiele für den body währen der h1 Tag, der die Überschrift darstellt oder das p Tag das zur Darstellung von Paragraphen da ist. Hier gibt es noch etliche Weitere HTMl Tags wie z.B. Formulare, Listen, Buttons, usw.

<html>

  <head>
  <!-- Head content -->
  </head>

  <body>
  <!-- Body content -->
  </body>
</html>

CSS

CSS ist eine Computersprache um Websites aufzubauen und zu struckturieren. Diese Sprache enthält Codeelemente und setzt sich aus diesen "Cascading stylesheets" zusammen, die auch CSS-Dateien (.css) genannt werden.

Es gibt ID's und Klassen um die Element im Dokument zu Identifizieren. ID's sind dazu da, um einzellne Elemente zu bestimmen:

#beispiel {}

während Klassen dafür da sind, mehrere Elementen zu bestimmen.

.beispiel {}

Man kann auch alle Elemente eines Tags (z.B. alle divs) bestimmen.

div {}

Man dies auch Kombinieren.

div.beispiel {}
.beispiel#beispiel {}

JavaScript

JavaScript gehört heutzutage mit zu den Kernkomponent einer Website. Es gibt diverse Anwendungsgebiet hierfür wie z.B. Client-Seitiges Javascript um die Website Interaktiver zu gestallten oder Element dynamisch nach zu laden.

Um ein paar bekannte Einsatzmöglichkeiten, die jeder kennt, zu nennen wären hier Seiten die "infinity scrolling" nutzen. Der Benutzer scroll die Seite und es werden kontinuierlich Element wie z.B. Bilder nachgeladen.

Andere einsatztgebiete sind Lightboxen, Slider oder Filter.

In gegensatzt zu CSS und HTML ist JavaScript eine vollwärtige Programmiersprache.

// Eine simple for Schleife
for(const i = 0; i < 10; i++) {
  console.log(i)
}

// Ein If Conditional
const x = 4
if(x === 4) {
  console.log("x is gleich 4")
} else {
  console.log("x is ungleich 4")
}
